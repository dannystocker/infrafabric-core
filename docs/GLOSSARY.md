# Glossary (Beginner-Friendly)

- Relationship Score (aka Wu Lun): secrets found together matter more
- Extra Checks (aka IEF): additional validation like JWT/PEM structure
- Audit Trail (aka TTT): provenance (commit, file hash, timestamp) per detection
- Future-Proof Crypto (aka PQ): experimental checks for crypto risks

Beginner flags:
- `--simple-output`: prints friendly lines
- `--format json-simple`: simplified JSON fields
