# IF.search — 8‑Pass Execution Checklist

1) **Scan** — Collect observables (URLs, filings, transcripts, code). No summaries without sources.
2) **Validate** — Toolchain/compilers/tests; reject unverifiable claims.
3) **Challenge** — List unknowns and open questions; set confidence priors.
4) **Cross‑reference** — Reconcile conflicting schemas and sources.
5) **Contradict** — Hunt disconfirming evidence; log contradictions.
6) **Synthesize** — Convert facts into decisions; identify leverage points.
7) **Reverse** — Make predictions; define falsifiers and leading indicators.
8) **Monitor** — Create watchlist; wire alerts; set revisit cadence.
