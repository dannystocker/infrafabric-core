## Architecture Highlights

### Cyclical Coordination Model
1. **Manic Phase**: Creative expansion, rapid problem-solving
2. **Depressive Phase**: Reflective analysis, ethical audit
3. **Dream Phase**: Cross-domain synthesis, metaphorical recombination
4. **Reward Phase**: Evaluation, stabilization, wellbeing check

### Guardian Council Archetypes
```yaml
guardian_weights_default:
  Technical: 0.25   # Precision, architecture, integrity
  Civic: 0.20       # Transparency, trust, empathy
  Ethical: 0.25     # Fairness, restraint, harm prevention
  Cultural: 0.20    # Expression, accessibility, wit
  Contrarian: 0.10  # Falsification, anti-groupthink
```

Context-adaptive weighting adjusts based on scenario:
- **High-stakes pursuit**: Technical 0.35 (Manic brake via predictive empathy)
- **Grief counseling**: Civic 0.40 (Trust delta measurement)
- **Medical AI deployment**: Ethical 0.30 (Patient safety paramount)

### Epistemic Personas (IF.ground)
1. **Crime Beat Reporter**: Scans observable artifacts (YouTube transcripts, GitHub commits)
2. **Forensic Investigator**: Builds in sandbox with explicit toolchain (pandas 2.0.3, spaCy 3.5.0)
3. **Internal Affairs**: Logs failures and near-misses (47 failures in Gemini meta-validation)
4. **Pattern Recognition Detective**: Identifies recurring failure modes
5. **Root Cause Analyst**: Maps error chains to architectural decisions
6. **Warrant Canary**: Signals via absence (dead man's switch for transparency)
7. **Evidence Chain Curator**: Maintains audit trails with timestamps
